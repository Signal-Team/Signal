# 📡 Signal

**마케팅 실무자를 위한 키워드 변화 추적 플랫폼**

> 2026 Blaybus 인사이트톤 출품작

<br />

## 🎯 서비스 소개

마케팅팀·전략기획팀 실무자들은 방대한 데이터 속에서 **맥락을 읽어내는 것**에 어려움을 겪습니다.
기존 데이터 수집 도구들은 *'무슨 일이 일어났는가'* 에만 집중할 뿐, 전략의 방향성을 제시하지 못합니다.

**Signal**은 사용자가 추적하고자 하는 키워드와 질문을 입력하면,
AI가 맞춤형으로 분석 결과를 제공하는 서비스입니다.

<br />

## ✨ 핵심 기능

### 1️⃣ 키워드 & 핵심 질문 입력
추적하고 싶은 키워드와 전략적 질문을 입력합니다.

```
키워드  : 영화, 굿즈 콤보, 팝콘 세트, 한정판, 프로모션
질문    : 프로모션으로 영화관 굿즈(한정판·콜라보·소장형)를 내면 실제 수요가 있을까?
         수요가 있다면 어떤 운영 방식(수량·배포·기간·채널)이 성공 확률이 높은가?
```

### 2️⃣ 추적 설정
추적 목적, 카테고리, 업데이트 주기를 설정합니다.

- **카테고리** — 음악 · 패션 · 요리 · 기술 · 여행 · 건강 · 교육 · 영화 · 기타
- **업데이트 주기** — 6시간 / 12시간 / 24시간(권장)

### 3️⃣ AI 분석 대시보드
AI가 웹에서 수집·분석한 데이터를 기반으로 대시보드를 생성합니다.

| 항목 | 내용 |
|------|------|
| 🔍 AI 한줄 요약 | 핵심 인사이트를 한 문장으로 요약 |
| 📝 상세 분석 답변 | 질문에 대한 심층 분석 결과 |
| 📅 타임라인 & 출처 | 시간순 변화 흐름과 데이터 출처 |

### 4️⃣ 키워드 트래킹 업데이트
설정한 주기에 따라 키워드 트래킹이 **자동으로 업데이트**됩니다.
트렌드의 변화를 놓치지 않고 실시간으로 감지할 수 있습니다.

### 5️⃣ 보고서 생성
분석 결과를 보고서 형태로 출력하여 실무에서 즉시 활용 가능합니다.

<br />

## 💡 Signal만의 강점

> 단순히 데이터를 보여주는 것이 아니라, **전략의 방향성**을 제시합니다.

| 강점 | 설명 |
|------|------|
| 🤖 AI 전략 질문 | 단순 데이터 분석을 넘어 AI에게 직접 전략 방향성을 질문하고 답변 수령 |
| 📈 실시간 트렌드 감지 | 과거 데이터 + 주기적 업데이트로 변화하는 트렌드를 지속 감지 |
| 📄 보고서 출력 | 정리된 분석 결과를 보고서 형식으로 출력하여 실무 즉시 활용 가능 |

## 🚀 시작하기

**1. 의존성 설치**
```bash
npm install
```

**2. 환경 변수 설정**
```bash
cp .env.example .env.local
```

`.env.local` 파일에 아래 값을 입력합니다.
```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
ANTHROPIC_API_KEY=
```

**3. 개발 서버 실행**
```bash
npm run dev
```
